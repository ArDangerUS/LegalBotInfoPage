Основные возможности

Полный спектр юридических услуг в вашем смартфоне

🤖

AI-консультации

Мгновенные ответы на юридические вопросы с использованием GPT-4.1 и o3-mini. Анализ законодательства РФ, судебной практики и правовых норм.

📄

Генерация документов

Автоматическое создание договоров, заявлений, исков, претензий и других документов по ГОСТ Р 7.0.97–2016 с учетом всех требований.

📊

Анализ документов

Загрузка и анализ PDF, DOCX, TXT файлов. Выявление юридических рисков, проверка соответствия законодательству.

💳

Система подписок

Интеграция с ЮKassa для автоматических платежей. Гибкие тарифные планы с автопродлением и управлением подписками.

💾

История диалогов

Сохранение всех консультаций и документов. Возможность продолжить диалог с сохранением контекста.

🔒

Защита данных

Соответствие ФЗ-152 "О персональных данных". Шифрование, безопасное хранение в Supabase и Яндекс.Облаке.

Ключевые особенности проекта

Что делает наш бот уникальным

🎯 Специализированные промпты

Система включает более 20 специализированных промптов для различных типов документов: договоры, иски, претензии, заявления. Каждый промпт оптимизирован под требования российского законодательства и ГОСТ. 

CONTRACT_PROMPT
APPLICATION_PROMPT
CLAIM_PROMPT
PRETENSE_PROMPT
POWER_OF_ATTORNEY_PROMPT

⚡ Асинхронная архитектура

Полностью асинхронная обработка запросов позволяет боту одновременно обслуживать сотни пользователей без задержек. Payment Monitor работает в фоновом режиме, автоматически проверяя статус платежей. 

async def handle_message()
async def process_payment()
async def generate_document()
asyncio.create_task()

🔄 Гибридная БД

Миграция с MongoDB на Supabase PostgreSQL с сохранением обратной совместимости. Автоматическая синхронизация данных, REST API для быстрого доступа, real-time подписки на изменения.
